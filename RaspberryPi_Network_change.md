
# 라즈베리파이에 연결된 네트워크 바꾸는 법

(마이크로 SD카드 새로 안 구워도 됨) 

## 1. 네트워크 설정 파일 찾아서 열기

터미널에서 아래 명령어를 입력하여 설정 파일을 엽니다. 

```bash
sudo nano /etc/netplan/50-cloud-init.yaml

```

만약 해당 파일이 없다면, 아래 명령어로 파일 위치를 확인하세요. 

```bash
ls /etc/netplan/

```

## 2. 네트워크 정보 변경

해당 `.yaml` 파일에서 `access-points`의 **이름(SSID)**과 **비밀번호**를 연결하고자 하는 새 네트워크의 것으로 변경합니다. 

* 처음 변경 시 비밀번호가 암호화되어 있을 수 있으나, 수정할 때는 **평문**으로 작성해도 무방합니다. 


* 
**주의:** YAML 파일은 들여쓰기에 매우 민감하므로 간격을 정확히 맞춰야 합니다. 



**작성 예시:**

```yaml
network:
    ethernets:
        eth0:
            dhcp4: true
            optional: true
    version: 2
    wifis:
        wlan0:
            access-points:
                "새로운_와이파이_이름":
                    password: "새로운_비밀번호"
            dhcp4: true

```



## 3. 설정 적용하기

수정이 완료되면 아래 명령어로 설정을 적용합니다. 

```bash
sudo netplan apply

```

* 이 명령어를 실행하면 기존 네트워크 연결이 끊어지므로, 사용하던 터미널(CMD) 창이 **먹통 상태**가 됩니다. 


* 
`WARNING` 메시지가 뜬다면 적용이 잘 된 것이므로, 해당 창을 닫고 새 터미널을 엽니다. 



## 4. 다시 SSH 접속

1. 노트북(PC)의 네트워크를 라즈베리파이에서 변경한 **새 네트워크와 동일하게 변경**합니다. 


2. SSH를 통해 라즈베리파이에 다시 접속합니다. 


3. 설정이 성공했다면 일반적인 접속과 동일하게 비밀번호 입력 창이 뜹니다. 



> **⚠️ 주의사항 (핫스팟 사용 시)**
> 휴대폰 핫스팟을 이용하는 경우, 밴드를 **2.4GHz**를 지원하도록 설정해야 합니다. **5GHz**로 설정할 경우 라즈베리파이가 인식하지 못할 수 있습니다.
